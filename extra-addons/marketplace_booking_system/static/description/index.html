Odoo Marketplace Booking & Reservation Management

Odoo Marketplace Booking and Reservation Management module allows to create and manage time-based services. The admin and sellers can create booking product which acts as time-based services on the website and can be used to book an appointment or reservation.
The sellers can define start and end date, add time slots and plans available for the booking product. The customers can click on the product, choose from the date and time slots available, select plans and place an order in Odoo website to book their slot. Once the order is confirmed, the slot is booked.

Basic Features

  • The module allows the customers to book appointments and reservations using Odoo website.
  • The customers can choose the available date, time slots and the number of bookings as per his choice and place the order to confirm his booking/reservation.
  • The admin and sellers can create booking product which acts as time-based services on the website and can be used to book an appointment or reservation.
  • The admin and sellers can define start and end date, add time slots and plans available for the booking product.
  • The admin and sellers can set the maximum limit for the number of bookings/reservations for an order.
  • The sellers can create different timeslots and booking plans and request to admin for their approval.
  • Admin can configure the auto approval settings of booking plans and timeslots.
  • Sellers can check their own booking orders at the backend.

Website View of Odoo Marketplace Booking & Reservation Management

Booking Product

The booking product shows up on the Odoo website along with other product.
Click on the 'BOOK NOW' button to book the appointment or make a reservation.
Select from the available dates. Then select the desired time slot according to the availability.
Choose your plan- Gold, Silver, Platinum.
Select the number of slots ( Number of slots for one order depends upon the Maximum allowed).

Checkout Page

The Odoo website checkout page shows the details of the booking with the product in the order lines.
Seller View of Odoo Marketplace Booking & Reservation Management

The sellers can create a service type products- 'Available For booking' and define start date and end date, timeslots, booking plans and available quantities, etc in the booking product. These products appear as a time-based service product.

A seller can check all the booking orders by applying this filter.

A seller can create a new timeslot and request to admin for approval.

A seller can view all the booking plan, can create a new and request to admin for approval.

Admin View of Odoo Marketplace Booking & Reservation Management

Admin can check all the booking products of marketplace sellers.

Admin can check all the booking orders of marketplace sellers.

Admin can check all the booking timeslots and approve/reject them.

Admin can check all the booking plans and approve/reject them.

Configuration View of Odoo Marketplace Booking & Reservation Management

The admin can set default configuration settings for the auto approval of booking timeslots and plans.
